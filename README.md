# 🚀 DevForge AI

DevForge AI is a smart developer tool that generates **project ideas** and recommends **relevant GitHub repositories** based on your skills and interests.

Instead of searching the internet for hours, DevForge AI helps developers quickly find project ideas and real open-source repositories to start building immediately.

## ✨ Features

- 💡 AI-based project idea generation
- 🔎 GitHub repository recommendations
- 🌍 Works for multiple domains (AI, ML, Gaming, Finance, Healthcare, Agriculture)
- 🎨 Clean interactive UI built with Streamlit
- ⚡ Fast and lightweight


## 📌 Example

Input

Skills: python pandas  
Interest: healthcare

Output

Project Idea:
Healthcare Disease Prediction System

Recommended GitHub Repositories:
• disease-prediction-ml  
• medical-diagnosis-ai  
• healthcare-analytics-python

## ⚙️ Installation

Clone the repository

git clone https://github.com/deaththekid1614/devforge-ai.git

Go into the folder

cd devforge-ai

Create virtual environment

python3 -m venv venv

Activate environment

source venv/bin/activate

Install dependencies

pip install -r requirements.txt

Run the app

streamlit run app/app.py
